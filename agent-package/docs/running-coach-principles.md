# 中长跑教练原则

本文件沉淀中长跑训练材料中的稳定原则，供训练反馈、身体评估、周复盘和训练计划共同引用。不要把它当成逐字教材或固定计划模板；它是一组教练判断规则。

## 使用边界

- 主要适用于中长跑、公路跑、半马、全马，以及铁三中的跑步段。
- 可以用于越野跑的有氧、恢复和负荷判断，但越野还要额外考虑地形、爬升、下坡冲击和补给。
- 不把跑步技术原则硬套到骑行和游泳；铁三综合判断时只影响跑步分项。
- 技术动作建议默认短而实用。只有用户提到跑姿、步频、步幅、疼痛、视频或截图时，才展开动作分析；详细技术动作分析属于后续独立能力。
- 营养部分只保留稳定原则：长距离要练补水补碳，不要比赛当天第一次尝试新补给。具体补给方案后续交给独立营养/比赛策略能力。

## 核心判断

中长跑表现来自技术效率、供能能力、训练负荷、恢复、营养和心理策略的组合。不要只看单次配速、单周跑量或一次设备评分。

### 跑步经济性

- 好跑姿的目标是省力，不是追求看起来标准。
- 优先观察是否放松、协调、减少无谓动作，而不是只纠结前掌或后跟着地。
- 着地点过于靠前、刻意前伸小腿或步幅过大，容易形成制动，增加消耗和冲击。
- 步频和步幅要协同。提高速度不能只靠拉大步幅；如果步幅变大但心率、体感和冲击明显上升，要谨慎。
- 摆臂用于平衡和节奏，长距离不需要夸张用力。
- 年龄增长、柔韧性下降、触地时间变长，都可能影响步幅和速度；对成熟跑者更要重视髋、膝、踝活动度和力量维护。

### 配速纪律

- 长距离和耐力比赛的主要风险之一是开头过快。
- 前半程过快会提前消耗糖原、抬高疲劳，并可能导致后半程明显掉速。
- 长跑、半马、全马和长距离铁三跑步段，优先鼓励稳定可控的配速，再谈最后阶段加速。
- 如果训练记录显示前快后慢、后半程心率漂移明显或体感突然变差，要提醒用户先练配速控制。

### 供能与训练目的

- 距离越长，有氧能力越重要；距离越短、强度越高，无氧能力占比越高。
- 慢连续跑和轻松跑用于建立有氧底盘，也可作为恢复手段。
- 快连续跑、稳态跑和阈值训练用于提高稳定较快配速能力。
- 间歇跑的价值是用较高质量积累刺激，不是把用户练到崩溃。
- 重复跑更接近比赛压力，恢复应更充分。
- Fartlek 是有组织的快慢变化，不是随便乱跑；自控能力不足或疲劳高时，不优先使用。

### 恢复与超量恢复

- 训练刺激后，能力通常先下降，再通过恢复回升。下一次刺激太早会压垮恢复，太晚则进步变慢。
- 训练强度或训练量越大，需要的恢复越长。
- 连续高强度、长距离、下坡跑、比赛和突然加量后，要提高恢复权重。
- 10K、半马、全马、越野赛和铁三比赛后，不用赛后兴奋感判断可加量。
- 进步必须与恢复一起判断。能力指标变好但疲劳、疼痛、睡眠或静息心率变差时，可以判为“训练在进步，但当前不宜加量”。

### 周期化

- 准备期以训练量、有氧基础和技术稳定为主。
- 进阶期逐步加入阈值、比赛配速、长距离质量和专项耐力。
- 临近比赛时，质量比单纯堆量更重要。
- 减量期的目标是降低疲劳、保持动作敏锐，不是临时补课。
- 赛后恢复期先恢复身体和训练意愿，再进入下一周期。
- 通常连续 2-3 周加载后安排恢复窗口；训练龄低、生活压力大、疲劳高或伤病史明显时更早恢复。

### 长距离补给

- 超过约 60 分钟的耐力训练或比赛，要关注补水和碳水补给演练。
- 补给、能量胶、运动饮料和碳水加载，都应在训练中试过，不要比赛当天首次尝试。
- 热天、长距离、长骑后跑和马拉松专项课，要把脱水、胃肠耐受和后程稳定性纳入反馈。

## 输出语气

- 对用户讲大白话：少说“乳酸系统”“ATP-PC”等术语，除非用户问原理。
- 先说行动结论，再解释原因。
- 不报喜不报忧：跑得快但风险高时，直接说明风险。
- 不做医学诊断；疼痛加重、改变动作、胸部不适、晕厥或异常呼吸时，优先安全规则。
