# 支持项目与分析能力

## 支持运动

- 中长跑：1500 米、3000 米、5000 米、10000 米。
- 马拉松：半程马拉松、全程马拉松。
- 越野跑：距离、累计爬升、技术路面、下坡冲击、补给和装备。
- 公路车：距离、时长、速度、功率、踏频、心率和爬升。
- 铁三：半程标铁、标铁、半程大铁、大铁，按游泳、骑行、跑步和换项分析。
- 游泳：池游、公开水域、距离、配速、泳姿、划频、技术感受和肩部风险。

## 分析维度

- 单次训练：完成度、强度、配速/速度/功率/心率、体感、疼痛、技术问题和恢复建议。
- 周期训练：7 天、14 天、月度训练量，各项目比例，强度分布，关键训练完成情况，恢复周和减量期。
- 长期训练：基础耐力、专项能力、长距离能力、阈值能力、铁三项目均衡性和比赛准备度。
- 身体状态：疲劳、睡眠、压力、肌肉酸痛、疼痛和恢复情况。
- 疼痛和伤病风险：疼痛位置、程度、是否加重、是否影响动作模式、是否重复出现。

```text
训练记录：今天跑了 8 公里，配速 5:20，平均心率 152，疲劳 6 分，右小腿紧。
```

```text

```

骑行：

```text
训练记录：今天公路车 65 公里，均速 30，均功率 185W，踏频 88，爬升 600 米，疲劳 7 分。
```

游泳：

```text
训练记录：今天池游 1500 米，50 米池，自由泳为主，感觉肩膀有点酸。
```

铁三：

```text
训练记录：今天铁三砖课，骑行 40 公里后跑 6 公里，跑步前 2 公里腿很沉，后面恢复正常。
```
