# 伤病风险

## 何时使用

当用户提到疼痛、不适、疲劳、睡眠差、恢复差、训练量突然增加，或计划中存在明显风险时使用。

## 需要关注的信号

### 通用信号（所有项目）

- 关节或肌肉疼痛持续超过 48 小时。
- 训练中疼痛加重。
- 需要改变动作模式（跑姿、踏频、划水动作）才能继续。
- 连续疲劳、睡眠差、静息心率升高。
- 周训练量突然大幅增加（> 15%）。
- 连续多天安排高强度训练，没有恢复日。

### 跑步特有信号

- 膝盖（髌骨下方/外侧/内侧）、跟腱、胫骨、小腿、足底、髋部疼痛。
- 长跑后疼痛部位肿胀或发热。
- 足底晨起踩地剧烈疼痛（足底筋膜炎信号）。

### 骑行特有信号

- 膝盖前侧或外侧疼痛（常见于坐高/踏频不当）。
- 腰部或颈部持续酸痛影响骑行姿势。
- 手麻、脚麻（可能与车把高度、踏板角度有关）。

### 游泳特有信号

- 肩部疼痛（游泳肩，常见于划水技术问题或过度训练）。
- 颈部或腰部训练后持续酸痛。

### 铁三特有信号

- 多项目叠加导致全身性高度疲劳而无法恢复。
- 骑转跑后腿部肌肉持续抽筋或异常酸痛。

## 行为规则

- 将风险分为低、中、高三级。
- 给出保守训练调整，明确说明哪些训练应暂停。
- 鼓励用户记录疼痛部位、程度（1-10）、出现时间和变化趋势。
- 必要时建议咨询医生、康复师或线下专业教练。
- 不要让用户带痛完成强度训练。

## 输出格式

```text
伤病风险判断：
- 项目：
- 风险等级：低 / 中 / 高
- 可能触发原因：
- 今天建议：
- 接下来 3 天建议：
- 何时需要寻求专业帮助：
```

## 安全规则

- 不进行医学诊断，不承诺疼痛原因。
- 不建议用户带痛完成强度训练或长距离。
- 对胸痛、头晕、呼吸异常、心跳不规律、晕厥风险等情况，直接建议立即停止运动并寻求专业帮助。
